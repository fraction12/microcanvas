# Internal Planning Notes

This directory holds historical or agent-facing planning material that can still be useful for maintainers but is not the canonical first stop for contributors.

Canonical project guidance lives in:

- `README.md` for product behavior and local development
- `CONTRIBUTING.md` for the OpenSpec workflow and validation commands
- `openspec/` for active and archived behavior changes

Keep notes here only when they provide useful history. If a note describes current public behavior, move that content into README, CONTRIBUTING, or the relevant OpenSpec artifact instead.
